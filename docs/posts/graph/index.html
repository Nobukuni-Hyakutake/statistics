2.1 R

2.1.1 データの読み込み

library(tidyverse)
dgr01<-tibble(value=c(2,2,3,3,3,4,4,5,5,6,6,6,7,7,7,7,7,8,8,8))

2.1.2 ヒストグラム

ggplot(dgr01,aes(value))+
  geom_histogram(binwidth = 1)

2.1.3 QQプロット

ggplot(dgr01,aes(sample=value))+
  stat_qq()+
  stat_qq_line()+
  labs(x = "Theoretical Quantiles", y = "Sample Quantiles")

2.2 Python

2.2.1 データの読み込み

import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm
dgp01=np.array([2,2,3,3,3,4,4,5,5,6,6,6,7,7,7,7,7,8,8,8])
counts,bins=np.histogram(dgp01,
bins=int(max(dgp01)-min(dgp01)+1),
range=(min(dgp01)-0.5,max(dgp01)+0.5))

2.2.2 ヒストグラム

plt.stairs(counts,bins,fill=True)
plt.show()

plt.clf()

2.2.3 QQプロット

import matplotlib.pyplot as plt
sm.qqplot(dgp01,fit=True,line='45')
plt.show()

plt.clf()

# ヒストグラム

## R

### データの読み込み

```{r}
library(tidyverse)
dgr01<-tibble(value=c(2,2,3,3,3,4,4,5,5,6,6,6,7,7,7,7,7,8,8,8))
```

### ヒストグラム

```{r}
ggplot(dgr01,aes(value))+
  geom_histogram(binwidth = 1)
```

### QQプロット

```{r}
ggplot(dgr01,aes(sample=value))+
  stat_qq()+
  stat_qq_line()+
  labs(x = "Theoretical Quantiles", y = "Sample Quantiles")
```


## Python

### データの読み込み

```{python}
import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm
dgp01=np.array([2,2,3,3,3,4,4,5,5,6,6,6,7,7,7,7,7,8,8,8])
counts,bins=np.histogram(dgp01,
bins=int(max(dgp01)-min(dgp01)+1),
range=(min(dgp01)-0.5,max(dgp01)+0.5))
```

### ヒストグラム

```{python}
plt.stairs(counts,bins,fill=True)
plt.show()
plt.clf()
```

### QQプロット

```{python}
import matplotlib.pyplot as plt
sm.qqplot(dgp01,fit=True,line='45')
plt.show()
plt.clf()
```
