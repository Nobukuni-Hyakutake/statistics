# 目的

RとPythonでグラフ作成をします。

# ヒストグラム

## R

### データの読み込み

```{r}
library(tidyverse)
dgr01<-c(2,2,3,3,3,4,4,5,5,6,6,6,7,7,7,7,7,8,8,8)
dgr02<-tibble(value=dgr01)
```

### グラフ化

```{r}
ggplot(dgr02,aes(value))+
  geom_histogram(binwidth = 1)
```


## Python

### データの読み込み

```{python}
import numpy as np
import matplotlib.pyplot as plt
dgp01=[2,2,3,3,3,4,4,5,5,6,6,6,7,7,7,7,7,8,8,8]
counts,bins=np.histogram(dgp01,bins=int(max(dgp01)-min(dgp01)+1),range=(min(dgp01)-0.5,max(dgp01)+0.5))
```

### グラフ化

```{python}
plt.stairs(counts,bins,fill=True)
plt.show()
```
